# Ctrl+Eat — Frontend Demo

Student meal delivery & subscription service frontend built with React, Vite, and Tailwind CSS.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page — hero, mood cards, feature bar |
| `/meal-plans` | Subscription pricing page |
| `/dashboard` | User dashboard with orders & stats |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Stack

- **React 19** + TypeScript
- **Vite** for dev/build
- **Tailwind CSS v4** for styling
- **React Router** for navigation
- **Lucide React** for icons
